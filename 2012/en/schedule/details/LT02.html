Title

Generating excel data with charts, formatting and more with the axlsx gem

Abstract

This presentation will explain why you should be using axlsx instead of CSV for your client reporting as well as showing some code examples of how it can be done. I'll be introducing axlsx. It is a new gem that lets Ruby and Ruby on Rails developers generate ECMA-376 standard compliant OOXML (xlsx) documents with charts, images, conditional formatting, custom styles, as well as interoperability with other spreadsheet programs like Numbers, LibraOffice, and Google Spreadsheet.
